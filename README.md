# Data Analytics Project

## Overview

This project demonstrates a complete Data Analytics workflow, starting from raw data extraction to business insights and visualization. The project includes data loading in Python, Exploratory Data Analysis (EDA), data cleaning, SQL analysis using PostgreSQL/MySQL/SQL Server, dashboard development in Power BI, report creation, and presentation preparation using Gamma.

The objective of this project is to transform raw data into meaningful insights that support data-driven decision-making.

## How to Run

### Prerequisites

* Python 3.x
* Jupyter Notebook
* PostgreSQL/MySQL/SQL Server
* Power BI Desktop

### Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/data-analytics-project.git
```

2. Install required Python libraries:

```bash
pip install pandas numpy matplotlib seaborn
```

3. Open Jupyter Notebook and run the notebooks.

4. Import the dataset into your preferred SQL database.

5. Execute SQL scripts from the SQL folder.

6. Open the Power BI dashboard file (.pbix).

7. Review the report and presentation files.
